# Term Translation

| Technical term | Plain-language translation |
|---|---|
| UI | 화면, 사용자가 보는 부분 |
| Server | 처리 규칙, 운영실 |
| Package / Shared | 공통 부품 |
| CLI | 실행/관리 도구, 관리자 콘솔 |
| Test | 검증 장치, 검사팀 |
| Route / API | 요청을 받는 창구 |
| State | 현재 기억하고 있는 값 |
| Component | 화면 조각 |
| Refactor | 겉동작은 유지하면서 내부 정리 |
| Dependency | 기대고 있는 도구/부품 |
